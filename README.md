# Auto Atende Suite

Complete AI customer service system for Brazilian auto repair shops, autocenters, and dealerships — from first WhatsApp message to loyal returning customer.

## Install

```bash
npx squads add claudio_felix/auto-atende-suite -y
```

## What's included

- **Auto Consultor** (agent) — strategic advisor that diagnoses the shop and recommends the ideal setup
- **Gerar Fluxo de Atendimento** (task) — generates complete WhatsApp intake flow with CARRO qualification
- **Gerar Sequência Pós-Venda** (task) — generates D+0 to D+365 retention sequence with NPS and seasonal campaigns
- **Jornada Cliente Completa** (workflow) — end-to-end orchestration from first contact to loyal customer

## Target market

Brazilian automotive sector. All generated scripts are in Portuguese (pt-BR).

## Requirements

- Claude Code, Gemini CLI, or Codex
- WhatsApp Business API or partner platform (Z-API, Evolution API, BotConversa)
